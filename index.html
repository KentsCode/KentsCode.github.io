Daily Special

Description

Daily Special is a web application that allows customers to find the best specials in their area. Restaurants and bars have the ability to create a login and create new specials or delete specials. The login is handled through Auth0 and tied into their browser through a token for identification.


The database uses two collections to store data. One collection to store restaurant information such as an id, name, address and the token that ties them to their Auth0 login. The other collection contains an entry for all the specials.

Get It Or Ship It

Description

Get It Or Ship It is a web application that allows you to compare an online retailer with a brick and mortar retailer and gives you the full picture of your shopping experiences. It compares not only the cost of the item but also shipping price and drive time.

This application utilizes 3rd party APIs from Ebay and Wal-Mart to get product data, prices and the location of the nearest Wal-Mart with AJAX calls. An additional AJAX call is made to the Google Maps API for driving distance and directions. The site was built with with JQuery and Bootstrap, in addition to Javascript, HTML and CSS.

This was a group project from Bootcamp. We will be working on this again in the near future and giving it to a new URL
